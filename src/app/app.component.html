Viewer Application in Angular 7

  • Fetch query params
  • Authenticate User and if authenticated, get image information if belongs to that
  • Send schema request for the image
  • Open OpenSeadragon Viewer with image and tools
  • Pathpresenter Theme

Components Needed for this application

  • User Authentication Service
  • Image related Authorization and tile, schema requests Service
  • OpenSeadragon Viewer Component with zoom, full screen, roi change options
  • OSD share service
  • Annotation Component
  • Marking Tool Component
  • Viewer
